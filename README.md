# 02.02_exam
WPF приложение для расчета стоимости покупки жалюзи. Позволяет рассчитать стоимость по размерам и материалу, сформировать и сохранить текстовый чек.

Функции приложения:

1. Расчет стоимости:
Ввод параметров: ширина, высота, количество, материал
Автоматический расчет по формуле
Цены материалов:
Алюминий: 15.50 рублей за м2
Пластик: 9.90 рублей за м2

2. Формирование квитанции:
Генерация текстового чека по шаблону
Автоматическое сохранение в файл
Формат имени файла: номер_дата_стоимость.txt
Сохранение в папку "Чеки_жалюзи" в Документах

3. Тестирование:
Юнит-тесты для проверки расчетов

Технологии:
Язык: C#

Тестирование: MSTest
Среда разработки: Visual Studio 2022

Формула расчета:

Площадь = Ширина × Высота × Количество
Стоимость = Площадь × Цена_за_кв_метр

Цены:
- Алюминий: 15.50 руб/м²
- Пластик: 9.90 руб/м²
Формат чека:
Чек сохраняется в текстовом формате с следующей структурой:

ООО "Уютный Дом"
Добро пожаловать
ККМ 00075411     #3969
ИНН 1087746942040
ЭКЛЗ 3851495566
Чек №[номер]
[дата] [время] СИС.
====================================================================
жалюзи          [размер] м
материал        [материал]
====================================================================
Итог            =[стоимость]
Сдача           =0
Сумма итого:    =[стоимость]
Тестовые сценарии
Тест 1: Корректный расчет
Ввод: ширина=1.5, высота=4.0, количество=1, материал=Алюминий

Ожидаемый результат: 93.00 рублей
